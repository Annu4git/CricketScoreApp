## Important status codes :

### Success

- 200 - OK
- 201 - CREATED
- 202 - ACCEPTED

### Client Errors

- 400 - Bad Request
- 401 - Unauthorized (Actually unauthenticated)
- 403 - Forbidden
- 404 - Not Found

### Sever Errors

- 500 - Internal Server Error
- 502 - Bad Gateway
- 503 - Service Unavailable
- 504 - Gateway Timeout
